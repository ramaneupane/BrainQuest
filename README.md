# BrainQuest

A modern, responsive general knowledge quiz website designed for middle school students.

## Features

- Bright, colorful, kid-friendly interface
- Responsive layout for desktop, tablet, and mobile
- Category-based quiz selection
- 20-question random quizzes
- Difficulty filters
- Instant feedback and explanations
- Local storage for progress and quiz history
- Light and dark mode

## Run locally

Open index.html in your browser, or serve the folder with a simple static server such as:

```bash
python -m http.server 8000
```

Then visit http://localhost:8000.
